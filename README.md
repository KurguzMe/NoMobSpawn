# NoMobSpawn
**NoMobSpawn** — this is a lightweight Fabric mod that allows you to completely block specific mobs from spawning in your world.

---

## Download
The latest and most stable versions of the mod are always available on the official page:
* **[Download the mod on Modrinth](https://modrinth.com/mod/nomobspawn#download)**
* **[Download the mod on CurseForge](https://www.curseforge.com/minecraft/mc-mods/nomobspawn)**
